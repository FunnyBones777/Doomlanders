How to add these cards into EDOPro?

1. Go inside this folder (C:\ProjectIgnis\config) and open this file (configs.json) with a text editor.

2. Add this line into it:

   {
			"url": "https://github.com/FunnyBones777/Doomlanders",
			"repo_name": "Doomlander cards",
			"repo_path": "./repositories/funnybones777cards",
			"data_path": "expansions",
			"script_path": "script",
			"pics_path": "pics",
			"should_update": true,
			"should_read": true
   }

3. Place that below the "Project Ignis puzzles" line and you're done!
